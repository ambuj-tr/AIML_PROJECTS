# AIML_PROJECTS
A machine learning project that predicts the likelihood of heart disease using patient health data. Built using Python with data analysis, visualization, and Logistic Regression for accurate prediction.

🫀 Heart Disease Prediction using Machine Learning

📌 Overview
Heart disease is one of the leading causes of death worldwide.
This project uses machine learning to predict the likelihood of heart disease based on patient medical data.

🎯 Objective
To build an intelligent system that:

Analyzes patient health parameters
Identifies important patterns
Predicts whether a person has heart disease or not
💡 Features
📊 Exploratory Data Analysis (EDA)
📈 Data Visualization (Graphs & Heatmaps)
🧹 Data Preprocessing & Feature Engineering
🤖 Logistic Regression Model
📉 Model Evaluation (Accuracy, Confusion Matrix)
🌐 Optional Web App using Streamlit
🛠️ Tech Stack
Language: Python
Libraries: Pandas, NumPy, Matplotlib, Seaborn
Machine Learning: Scikit-learn (Logistic Regression)
Deployment: Streamlit (optional)
